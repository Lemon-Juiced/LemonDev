Qu Introduction

Qu is my third attempt at creating an Esoteric Programming Language.
Each release of Qu is indicated by the date it was released. The first release of Qu was on February 17th, 2024 and thus is 2024.02.17.

History

February 17th, 2024: Qu was first publicly released.
February 25th, 2024: Qu 2024.02.25 was released, which is its first Turing complete release.

Explanation

Qu is a language that is based on the concept of a queue as opposed to a stack.
The data in the queue can be manipulated by adding and removing data from the front of the queue using the provided instructions.

Compilation Instructions

If you have downloaded Qu from GitHub, there are two ways you can start.
If you would prefer to use the bundled "qu.exe" file you can skip steps 1 and 2.
These instructions use VSCode's notation, which is powered by PowerShell. If you are using another terminal, the notation may slightly differ.
These instructions use g++ as the compiler, if you have another compiler the notation may also change.
Step 1. From your terminal, cd .\dev\lemonjuice\qu\
Step 2. g++ .\qu.cpp -o qu .\error\errorHandler.cpp .\node\node.cpp .\operation\operationHandler.cpp
Note: The -o flag is optional, but it does enable compiler optimization, which is suggested.
Step 3. You can now run a .qu file with .\qu.exe {fileName}.qu".
If you need examples some exist on this page and in the GitHub repository.
