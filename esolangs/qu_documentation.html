Non-Queue Manipulation Instructions

Instruction Description Parameters Language Revision
PRINT Prints a given string or integer. 2024.02.20
QDISPLAY Prints the contents of the queue in a line seperated by commas. 2024.02.20

Queue Manipulation Instructions

Instruction Description Parameters Language Revision
EMPTY Empties the queue of all its elements. 2024.02.20
PEEK Peek and print the contents of the first element in the queue. 2024.02.20
PEEKLN Peek and print, with an end line character, the contents of the first element in the queue. 2024.02.20
POKE Randomize the order of the elements in the queue. 2024.02.20
POKE Randomize the order of the elements in the queue. 2024.02.20
POP Pop and print the contents of the first element in the queue. 2024.02.20
POPLN Pop and print, with an end line character, the contents of the first element in the queue. 2024.02.20
PUSH Push an element, that is a string or integer, into the queue. 2024.02.20
READ Reads an element specified by the user and pushes it into the queue. Required: READ - Just a line with no text.
Optional: READ {string/int} - Prints text on the line that matches the string or integer provided.		 2024.02.20
SORTDOWN Sorts the queue in descending order. 2024.02.20
SORTUP Sorts the queue in ascending order. 2024.02.20

Navigation Instructions

Instruction Description Parameters Language Revision
|{string}| Saves this location so that it can be reused in navigation during the run phase of the program. The string saved does not need to be encased in double quotes. 2024.02.20
GOTO Goes to a specified location in the program. GOTO requires a saved location or line index to follow it. 1. GOTO {int} - Goes to a specified line index.
2. GOTO {string} - Goes to a string that is associated with the saved location matching it's name.		 2024.02.20
